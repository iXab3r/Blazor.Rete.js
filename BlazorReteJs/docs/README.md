# BlazorReteJs Docs

This folder makes `Sources/BlazorReteJs` easier for AI agents and humans to
navigate.

## Five Passes

1. [retejs-ai-brief.md](retejs-ai-brief.md): very brief Rete skeleton, upstream
   documentation concepts, and useful example patterns.
2. [retejs-concepts.md](retejs-concepts.md): deeper Rete v2 concepts and
   extension principles for keeping bridge work aligned with upstream design.
3. [retejs-api-recipes.md](retejs-api-recipes.md): Rete v2 API details,
   recipes, and best practices from upstream docs and examples.
4. [retejs-ai-brief.md](retejs-ai-brief.md): Rete examples pass.
5. [blazor-retejs-technical-design.md](blazor-retejs-technical-design.md):
   current Blazor/C#/TypeScript adaptation.
6. Consumer-specific implementation notes live in their owning packages, not in
   this generic bridge folder.

## API Map

- [blazor-retejs-api.md](blazor-retejs-api.md): C# DTOs, TypeScript DTOs,
  facade operations, and build/test commands.
